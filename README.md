## 1. 프로젝트 개요
> '잠자리'는 국내 시장을 기반으로 한 개인과 개인간의 숙소 예약 사이트입니다. <br> 
코로나19의 영향으로 해외 여행 수요가 감소됨과 동시에 국내 여행 수요가 증가하였습니다. <br> 
기존 플랫폼인 에어비앤비의 약점을 보완하여 국내 지역 숙소 정보, 정확한 위치 정보, <br> 
사용자가 원하는 선별적인 정보를 제공하는 숙박 예약 사이트를 만들어 보고자 기획하게 되었습니다.

## 5. 구현기능
### 회원
- 로그인
- 회원가입
- 내 정보 조회 & 수정
- 비밀번호 변경
- 쪽지 목록 조회
- 쪽지 전송 & 삭제
- 과거 예약 내역
- 회원 탈퇴

### 숙소
- 숙소 등록
- 숙소 예약
- 숙소 수정
- 숙소 삭제
