## 1. 프로젝트 개요
>코로나19에 따른 해외 여행 수요 감소로 국내 여행 소요가 증가함에 따라 <br> 
국내 시장을 기반한 개인과 개인 간의 숙박 예약 사이트가 부족하여 <br>
국내 여행 시장에 알맞은 플랫폼을 개발하고자 하였습니다.
